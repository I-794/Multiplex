Number decoder

Read the system at a glance.

Odd one and two-digit numbers generally run north-south, even numbers generally run east-west, and three-digit auxiliaries inherit their final two digits from a parent route. Enter a designation to decode it.

⚠ The decoder reads the numbering convention. A number following the rules is not always an assigned, built route. Use the database to check status.
