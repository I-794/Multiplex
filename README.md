# Multiplex

A static, Vercel-ready Interstate database website.

## Local preview

Open `index.html` directly, or run a tiny local static server from this folder:

```powershell
python -m http.server 5173
```

Then visit `http://localhost:5173`.

## Vercel

This project does not require a build step. Import the repository in Vercel and leave the build command empty. Vercel can serve the root `index.html` as a static site.
